# Sourcing Checklist

Use this as a first-prototype purchasing filter.

## Search keywords

Chinese:
- 24V 高显色 COB 灯带 3000K
- 24V LED 灯带 CRI90 暖白
- LED 铝槽 乳白扩散罩
- 24V LED 调光器
- 24V 电源适配器 1A 2A
- M3 热熔铜螺母
- PETG 3D 打印耗材
English:
- 24V high CRI COB LED strip 3000K
- 24V CRI90 LED strip warm white
- LED aluminum channel opal diffuser
- 24V LED dimmer inline
- 24V DC adapter 1A 2A
- M3 heat set inserts
- PETG filament

## Before purchase

- Confirm voltage is 24V across LED strip, dimmer, and adapter.
- Confirm LED strip width fits aluminum channel.
- Confirm diffuser is opal/milky, not clear.
- Confirm adapter current rating exceeds LED strip requirement.
- Confirm dimmer supports the same voltage.
- Confirm cable connector type or plan to solder/terminate.
- Buy enough extra LED strip for one failed cut.
- Buy a small hardware kit rather than exact-count screws.
