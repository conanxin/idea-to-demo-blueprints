# First Prototype Assembly Runbook

## Goal

Assemble one ReadingCore-01 based prototype safely enough for functional observation and IDB-6E measurement.

## Steps

1. Review BOM and component specs.
2. Print the fit-test coupon.
3. Test M3 holes, diffuser slot, cable exit, and aluminum channel fit.
4. Adjust CAD if the coupon fails.
5. Print the full shell.
6. Cut LED strip to lamp-head length.
7. Attach LED strip to aluminum channel.
8. Install opal diffuser.
9. Mount ReadingCore-01 cassette into the shell.
10. Add strain relief at cable exit.
11. Attach arm and base.
12. Power on at low brightness first.
13. Run 10-minute observation.
14. Run IDB-6E lux grid.
15. Run 30-minute heat soak.
16. Run glare review.
17. Fill readiness report.
18. Decide: iterate / full prototype / stop.
