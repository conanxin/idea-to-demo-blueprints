# First Prototype Build Runbook

## Goal

Build one DIY Lamp Builder prototype using ReadingCore-01 and document enough evidence to decide whether to iterate the shell, the diffuser, the cable exit, the base, or the BOM.

## Inputs

- IDB-6G sourcing pack
- purchased prototype components
- IDB-6D fit-test coupon
- IDB-6E physical testing protocol
- selected shell .scad / STL
- printed shell, base, and arm mock

## Build phases

### 0. Pre-build check

- [ ] BOM components received
- [ ] 24V LED strip, dimmer, and adapter voltage match
- [ ] aluminum channel and diffuser fit together
- [ ] M3 hardware available
- [ ] cable strain relief planned
- [ ] fit-test coupon printed or ready to print
- [ ] measurement tools ready

### 1. Fit-test coupon

- [ ] print fit-test coupon
- [ ] test M3 holes
- [ ] test diffuser slot
- [ ] test aluminum channel clearance
- [ ] test cable exit
- [ ] record pass/fail

### 2. Full shell print

- [ ] choose print orientation from IDB-6D
- [ ] slice with prototype profile
- [ ] print full shell
- [ ] remove supports
- [ ] deburr sharp edges
- [ ] inspect warping and cracks

### 3. ReadingCore-01 cassette assembly

- [ ] cut LED strip to lamp-head length
- [ ] attach strip to aluminum channel
- [ ] install diffuser
- [ ] route wire through cable exit
- [ ] add strain relief
- [ ] mount cassette into shell

### 4. Base / arm assembly

- [ ] attach arm to shell
- [ ] attach base
- [ ] check stability
- [ ] check cable routing

### 5. First power-on

- [ ] power on at lowest brightness
- [ ] observe for 2 minutes
- [ ] check smell, flicker, heat, cable movement
- [ ] power off and inspect

### 6. Measurement

- [ ] lux grid
- [ ] 30-minute heat soak
- [ ] glare review
- [ ] readiness checklist

### 7. Report and iteration decision
- [ ] fill first-build report
- [ ] collect photos
- [ ] classify readiness
- [ ] decide next adjustment
