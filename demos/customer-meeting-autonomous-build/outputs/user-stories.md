# 用户故事：客户跟进看板

## US-001: 销售查看客户列表
- **作为** 销售
- **我想要** 在一个页面看到所有我负责的客户
- **以便** 快速决定今天跟进谁
- **验收标准**: 表格展示客户名称、阶段、风险、下次跟进时间
- **优先级**: 高

## US-002: 销售主管查看团队状态
- **作为** 销售主管
- **我想要** 看到团队所有客户的风险分布和跟进情况
- **以便** 及时发现风险并介入
- **验收标准**: 顶部统计卡片 + 风险颜色标识
- **优先级**: 高

## US-003: 销售更新跟进状态
- **作为** 销售
- **我想要** 更新客户的阶段和下次跟进时间
- **以便** 团队知道最新状态
- **验收标准**: 可编辑表格，更新后自动计算风险等级
- **优先级**: 中（第一版可只做展示）

## US-004: 销售记录客户备注
- **作为** 销售
- **我想要** 记录客户的关键信息（预算、决策人、竞争）
- **以便** 后续跟进有上下文
- **验收标准**: 备注列可查看
- **优先级**: 中
