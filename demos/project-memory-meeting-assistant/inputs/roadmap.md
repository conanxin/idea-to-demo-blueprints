# TeamFlow Dashboard — 路线图（Roadmap）

## Phase 1: MVP（当前）
- 基础任务看板（3 列：待办/进行中/已完成）
- 用户认证和权限（基础角色：管理员/成员）
- 简单风险登记（低/中/高三级）
- 会议记录模板

## Phase 2: 增强版（Q3 2026）
- 自定义工作流状态
- 甘特图视图
- 风险预警通知
- 集成飞书/Slack

## Phase 3: 企业版（Q4 2026）
- 多项目管理
- 高级权限系统（RBAC）
- 数据分析报表
- SSO 集成

## 已知约束
- 预算限制：不能引入昂贵的第三方服务
- 时间限制：MVP 必须在 6 月底前完成
- 人员限制：团队规模固定，不能扩招
