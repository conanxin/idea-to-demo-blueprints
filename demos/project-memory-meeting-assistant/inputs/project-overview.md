# TeamFlow Dashboard — 项目概览

## 项目定位

TeamFlow Dashboard 是一个团队项目管理看板，帮助中小团队追踪任务、风险、会议和项目阶段。

## 目标用户

- 研发团队（5-30 人）
- 项目经理
- 技术负责人

## 核心功能

1. **任务看板** — 拖拽式任务管理，支持自定义状态列
2. **风险追踪** — 识别、评估和监控项目风险
3. **会议管理** — 会议纪要、决策记录、行动项追踪
4. **项目阶段** — 清晰的项目里程碑和阶段划分

## 技术栈

- 前端：React + Tailwind CSS
- 后端：Node.js + Express
- 数据库：PostgreSQL
- 部署：Docker + AWS
