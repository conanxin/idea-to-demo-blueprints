DIY 可定制阅读台灯构建器

从一句话想法到可打印、可装配、可测试的实体阅读台灯原型。

v0.1.1-alpha IDB-6H.2

打开 Demo App 查看源码

Blueprint 摘要

把一个实体产品想法转化为可配置的阅读台灯 Demo：固定 24V 线性 ReadingCore-01 灯芯模块，用户定制外壳风格、颜色、纹样和刻字，并输出产品架构、配置 JSON、BOM 成本模型与制造计划。后续阶段逐步加入 CAD 导出、打印验证、物理测试、采购清单、真实组件研究和首台样机 build runbook。

核心假设

  • ReadingCore-01 固定：24V 线性 LED 灯芯，决定散热、光学和电气边界。
  • Shell 可变：风格、颜色、纹样、刻字由用户配置，但不可破坏核心 keepout。
  • 输出物：产品架构、配置 JSON、BOM、制造计划、装配步骤、CAD 导出、打印验证、物理测试、采购清单、供应商研究、build runbook。

使用流程

  1. 在 Demo App 输入一句话想法。
  2. AI 分析生成产品架构 6 字段。
  3. 在 Configurator 选择风格、颜色、纹样、刻字。
  4. 查看实时 SVG 预览。
  5. 生成配置 JSON、BOM 和装配步骤。
  6. 导出 OpenSCAD 文件进行 3D 打印。
  7. 执行打印验证、物理测试、采购和样机装配。

技术栈

  • 纯 HTML / CSS / JavaScript，无构建工具。
  • SVG 实时渲染外壳预览。
  • OpenSCAD 文本导出，可在本地生成 STL。

IDB-6C CAD Export Pass

IDB-6C 将外壳预览转化为可制造的 OpenSCAD 几何。它输出 shell_export.scad 文件，包含 core keepout、diffuser slot、mounting holes 和 cable exit，并提供一个手动 STL 导出命令。

字段说明
core_keepoutReadingCore-01 占用空间，避免外壳侵入。
diffuser_slot乳白扩散罩安装槽。
m3_mount_holes铝槽安装用 M3 孔位。
cable_exit线缆出口通道。
engraving用户刻字凹槽。

在 Demo 中打开 CAD Export

IDB-6D Print Validation Pass

IDB-6D 把 CAD 导出推进到打印验证。它包括打印方向建议、slicer 配置、fit-test coupon 和实测尺寸 log。

交付物用途
print-orientation-guide.md不同外壳的推荐打印方向。
slicer-profile-petg.iniPETG 原型打印参数。
fit-test-coupon.scad孔位、槽位、出口的装配测试件。
measured-fit-log.md实测尺寸与 CAD 偏差。

IDB-6E Physical Testing Pass

IDB-6E 增加实物测量流程：照度网格、30 分钟热稳定、眩光检查，并给出原型就绪 gate。

测试目标
Lux Grid阅读区 300-500 lux，重点区 500-750 lux。
Heat Soak30 分钟后外壳温升 ≤ 15°C。
Glare Review正常坐姿看不到直射 LED / 高亮点。
Readiness Gate通过全部测试后标记为 prototype-ready。

IDB-6F Build Package Pass

IDB-6F 将 DIY Lamp Builder 推进到第一台实体样机的采购与装配准备阶段。它提供 purchase-ready BOM、component spec sheet、sourcing checklist、quote comparison template、assembly runbook、supplier risk register 和 purchase decision gate。

模块用途
Purchase-ready BOM列出 24V LED 灯带、铝槽、扩散罩、电源、调光器、五金、打印材料和底座/支臂 mock。
Sourcing Checklist采购前检查电压、CRI、尺寸、调光器、电源电流和线缆应力释放。
Component Spec Sheet定义最低规格、推荐规格和禁止规格。
First Build Runbook从 fit-test coupon 到完整 shell 装配，再到 IDB-6E 实物测试。
Purchase Decision Gate确认是否可以买一套样机料，还是需要修改 BOM。

边界：IDB-6F 不是供应商背书，不是认证采购，也不是量产建议。真实 supplier shortlist 和实时价格对比留到 IDB-6G。

IDB-6G Real Component Research Pass

IDB-6G 将采购框架推进到真实组件候选研究，形成 first prototype 的 sourcing pack。它不是最终采购建议，也不是供应商背书。

交付物说明
real-component-candidates.csv/json记录 LED 灯带、铝槽、电源、调光器等真实候选件。
supplier-shortlist.md按组件类型整理候选供应渠道和风险。
purchase-search-keywords.md中英文搜索关键词。
sourcing-decision-matrix.csv用价格、规格可信度、匹配度、交付和风险打分。
first-prototype-shopping-list.md第一台样机的 shopping list。

边界：价格、库存、链接和安全标识都需要在下单前再次确认。真实 purchase execution 留到后续阶段。

IDB-6H First Prototype Build Pass

IDB-6H 将 sourcing pack 推进到首台样机 build day workflow：组件到货检查、fit-test coupon、完整外壳打印、ReadingCore-01 cassette 装配、首次通电观察、IDB-6E 测量，以及 first-build report。

交付物说明
first-build-runbook.md首台样机从到货到测试的完整装配流程。
first-build-report-template.md/json记录组件、fit-test、装配、测量和迭代决策。
assembly-photo-shot-list.md必须拍摄的装配证据清单。
troubleshooting-guide.mdM3 孔、扩散罩槽、发热、眩光、底座稳定等常见问题排查。
prototype-iteration-decision.md决定进入第二台样机、迭代 CAD、换 BOM 或停止重设计。

输出文件

  • blueprints.json — 全局 Blueprint 元数据。
  • demos/diy-lamp-builder/app/ — Demo App 源码。
  • demos/diy-lamp-builder/outputs/ — 设计、CAD、测试、采购、样机构建文档。

边界与免责

这是一个 maker prototype 教育 Demo，不是认证产品、电气安全审查、光学认证或生产采购建议。任何真实装配请自行评估风险，并在低压、有保护的条件下进行。

IDB-6I Idea-to-Physical-Product Compiler Pass

IDB-6I 把 DIY Lamp Builder 从单一灯具 demo 抽象成通用实体产品编译器模式。它定义了固定功能内核 + 可变外壳的结构、编译器 schema、复用 agent prompt、产品类型迁移矩阵和下一个 demo 候选。

交付物说明
idea-to-physical-product-compiler.md编译器定义和核心规则。
compiler-schema.json编译器输入输出 schema。
fixed-core-vs-variable-shell-pattern.md固定核心与可变外壳的分离模式。
product-type-adaptation-matrix.md不同产品类型的适配矩阵与风险。
reusable-agent-prompt.md复用 prompt，用于编译新产品想法。
next-product-candidates.md下一个产品候选和推荐顺序。

边界：IDB-6I 是方法抽象和原型模式，不是认证、量产平台或生产级制造系统。
