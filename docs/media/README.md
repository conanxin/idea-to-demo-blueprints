## 截图规范

如果补充截图，建议：

- **尺寸：** 1280×800 或 1440×900
- **格式：** PNG
- **完整截图：** 包含完整页面，不裁剪
- **标注：** 在 README 中引用时使用 `![描述](./media/<filename>.png)`
