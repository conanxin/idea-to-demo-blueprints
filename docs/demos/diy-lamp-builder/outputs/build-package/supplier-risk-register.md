# Supplier Risk Register

| Risk | Symptom | Likely Cause | Mitigation |
|---|---|---|---|
| CRI overclaim | book colors look dull | LED strip spec inaccurate | compare two suppliers, prefer known LED series |
| flicker | visible banding in phone slow-mo | poor dimmer/PWM | try another dimmer or fixed current mode |
| diffuser mismatch | diffuser does not snap in | channel/diffuser incompatible | buy channel and diffuser as a set |
| heat buildup | shell gets uncomfortable | LED power too high or weak aluminum path | reduce power, improve channel contact |
| cable strain | wire pulls loose | no grommet/strain relief | add cable clamp or printed relief |
| M3 fit issue | screws bind or wobble | printer hole shrinkage | use fit-test coupon and adjust hole diameter |
| warping | long shell lifts | poor bed adhesion/material settings | add brim, reduce stress, use PETG profile |
| engraving too small | text unreadable | FDM resolution limit | enlarge text or move to post-processing |
