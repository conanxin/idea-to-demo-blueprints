# Purchase Decision Gate

Before buying parts, confirm:

- [ ] LED strip, dimmer, and adapter are all 24V.
- [ ] LED strip width fits selected aluminum channel.
- [ ] Aluminum channel includes compatible opal diffuser.
- [ ] Adapter current rating is sufficient.
- [ ] Dimmer supports selected voltage/current.
- [ ] M3 hardware is available.
- [ ] Cable strain relief method is defined.
- [ ] PETG/PLA+ material is available.
- [ ] Fit-test coupon will be printed before full shell.
- [ ] No direct mains wiring is required.

Decision:

- [ ] Buy one prototype set.
- [ ] Revise BOM before purchase.
- [ ] Stop and redesign.
