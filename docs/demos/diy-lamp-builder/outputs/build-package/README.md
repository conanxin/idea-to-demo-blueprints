# IDB-6F Build Package

This package turns the DIY Lamp Builder demo into a first physical prototype preparation kit.

It does not claim certified sourcing or production readiness. It gives a practical purchase-and-build checklist for the first ReadingCore-01 desktop lamp prototype.

## What this package includes

- Purchase-ready BOM
- Component specification sheet
- Sourcing checklist
- First prototype assembly runbook
- Supplier risk register
- Quote comparison template
- Purchase decision gate
- First build issue log

## Recommended sequence

1. Review component-spec-sheet.md
2. Fill quote-comparison-template.csv
3. Confirm the purchase-decision-gate.md
4. Buy parts for one prototype only
5. Print the fit-test coupon
6. Print the first shell
7. Assemble the prototype
8. Run IDB-6E lux / heat / glare tests
