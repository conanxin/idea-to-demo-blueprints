# First Prototype Shopping List

Buy only enough for one prototype.

## Must buy

- [ ] 24V high-CRI LED strip, 3000K or 4000K
- [ ] Aluminum channel with opal diffuser
- [ ] 24V DC adapter, 1A-2A
- [ ] 12-24V inline dimmer
- [ ] M3 screws and heat-set inserts
- [ ] Cable strain relief / grommet
- [ ] PETG or PLA+ filament
- [ ] Base / arm mock parts

## Optional

- [ ] Second LED strip candidate for comparison
- [ ] Second dimmer candidate for flicker comparison
- [ ] Extra diffuser type
- [ ] Matte paint / sanding supplies

## Do not buy yet

- [ ] Large bulk LED reels
- [ ] Custom PCB
- [ ] Battery pack
- [ ] Mains-voltage driver
- [ ] Certification service
