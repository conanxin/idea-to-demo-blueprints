# IDB-6E Prototype Readiness Checklist

## Low voltage boundary

- [ ] Low-voltage adapter only
- [ ] No exposed conductor
- [ ] Strain relief at cable exit

## Mechanical fit

- [ ] ReadingCore-01 fits shell
- [ ] M3 holes fit
- [ ] Diffuser slides in
- [ ] Base stable

## Print quality

- [ ] No warping
- [ ] No layer crack near mount
- [ ] No sharp edges

## Thermal observation

- [ ] 30 min heat soak completed
- [ ] No shell softening or smell
- [ ] No discoloration

## Lux target

- [ ] Center >= 300 lux
- [ ] No reading-zone point below 200 lux
- [ ] Center-to-edge ratio reasonable

## Glare review

- [ ] Direct LED not visible
- [ ] No harsh reflection on glossy page
- [ ] Shadow acceptable
- [ ] Diffuser recessed

## Documentation

- [ ] Measurement log filled
- [ ] Heat notes recorded
- [ ] Glare notes recorded
- [ ] Next CAD adjustment recorded

## Ready / Iterate decision

Only proceed to full build when all critical checks pass. Otherwise iterate CAD and re-print.
