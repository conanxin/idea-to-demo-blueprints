# Next Product Candidates

After DIY Lamp Builder, the next demos should test whether the compiler generalizes.

## Candidate 1 — City Night Light
Fixed core:
- low-power LED night light module

Variable shell:
- city skyline
- architecture silhouette
- engraving
- translucent shell

Why next: close to lamp demo, lower power, lower thermal risk.

## Candidate 2 — Cultural Toothpick Box
Fixed core:
- drawer / storage module

Variable shell:
- temple / courtyard / architectural exterior

Why next: no electronics, tests mechanical fit and cultural shell.

## Candidate 3 — Desk Organizer
Fixed core:
- tray / grid module

Variable shell:
- personalized theme, name, pattern

Why next: safest physical product category, validates compiler beyond lighting.

## Candidate 4 — Aroma Lamp Shell
Fixed core:
- purchased aroma diffuser or low-heat module

Variable shell:
- decorative enclosure

Why later: heat / airflow risks require more validation.

## Recommended next demo

**City Night Light**, because it reuses lighting knowledge but simplifies reading-lamp performance requirements.
