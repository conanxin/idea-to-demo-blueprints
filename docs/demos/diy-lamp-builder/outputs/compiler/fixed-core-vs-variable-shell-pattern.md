# Fixed Core vs Variable Shell Pattern

## Pattern

A physical product can often be split into two parts:

- Fixed Core = risky, functional, electrical, mechanical, thermal, or safety-critical system
- Variable Shell = user-customizable form, color, theme, surface, ornament, enclosure, or interface layer

## Why it works

It keeps customization safe and manufacturable.

The user can change:

- style
- color
- pattern
- engraving
- decorative shell
- surface geometry
- exterior theme

The user cannot change:

- voltage
- thermal path
- mounting interface
- power system
- keepout volume
- mechanical safety boundary
- certified / purchased module
