# Build Day Checklist

## Tools

- [ ] 3D printer
- [ ] flush cutters / deburring tool
- [ ] small screwdriver
- [ ] soldering iron or connector kit
- [ ] multimeter
- [ ] timer
- [ ] lux meter or phone lux app
- [ ] infrared thermometer or contact thermometer
- [ ] phone/camera for photos

## Safety boundary

- [ ] low-voltage 24V only
- [ ] no direct mains wiring
- [ ] no exposed conductors
- [ ] cable strain relief present
- [ ] first power-on at low brightness
- [ ] stop if smell, smoke, softening, or uncomfortable heat appears

## Build order

- [ ] fit-test coupon
- [ ] full shell
- [ ] ReadingCore-01 cassette
- [ ] base / arm
- [ ] low-power observation
- [ ] IDB-6E measurement
- [ ] first-build report
