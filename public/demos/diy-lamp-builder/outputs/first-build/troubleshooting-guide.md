# Troubleshooting Guide
| Problem | Likely cause | Quick fix | Next CAD/BOM adjustment |
|---|---|---|---|
| M3 screw does not fit | FDM hole shrinkage | drill/ream lightly | increase hole diameter in test ladder |
| diffuser too tight | slot width too narrow | sand edge | widen diffuser slot |
| aluminum channel does not seat | CAD slot too narrow or channel different | sand slot | update channel clearance |
| LED dots visible | diffuser too clear or shallow channel | add diffuser layer | deeper channel or opal diffuser |
| shell gets hot | weak thermal path or high power | lower brightness | larger aluminum path / reduce LED power |
| flicker visible | poor PWM dimmer | try another dimmer | shortlist better controller |
| cable pulls out | no strain relief | add grommet/clamp | larger cable exit + relief feature |
| base tips over | center of mass too high | add weight | larger/heavier base |
| glare at eye position | diffuser exposed | rotate lamp or add visor | recess diffuser further |
| warped long shell | bed adhesion/stress | brim/temperature tuning | split shell or adjust orientation |
