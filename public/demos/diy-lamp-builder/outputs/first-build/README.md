# IDB-6H First Prototype Build Package

This package turns the DIY Lamp Builder sourcing pack into a first physical build workflow.

## Purpose

Guide the first actual prototype build day:

1. confirm purchased parts
2. print fit-test coupon
3. print shell
4. assemble ReadingCore-01 cassette
5. power on safely
6. capture build photos
7. run IDB-6E measurement tests
8. write first-build report
9. decide the next iteration

## Boundary

This is a maker prototype workflow. It is not electrical certification, photometric certification, thermal certification, or production QA.
