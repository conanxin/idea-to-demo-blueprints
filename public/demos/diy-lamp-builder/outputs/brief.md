# 输出 Brief（Agent 生成结果示例）

> 这一份是 Agent 按 `prompts/build-prompt.md` 处理"北京风格阅读台灯，外壳可定制，适合书桌使用"时产出的 Brief。

---

## 一句话定位

DIY Lamp Builder 把"一个实体灯具想法"在 5 分钟内转成"可演示的受控定制产品 Demo"——固定内核 ReadingCore-01 + 可换外壳 Shell，让独立开发者和文创团队能低成本验证产品可行性。

## 适用人群

- 独立开发者：想试水实体产品但不敢大批量打样
- 文创产品设计师：想做"有北京味儿"的灯具或文创周边
- 产品经理：跟客户解释"为什么这个灯值这个价"
- AI Agent 使用者：让 AI 把"想法"变成"产品 Demo"
- 硬件原型爱好者：按 JSON 自己找元器件打样

## 核心价值主张

1. **架构即哲学** —— ReadingCore-01 永远是同一个内核，外壳只承担造型
2. **结构正确** —— LED 灯带贴铝槽 + 乳白扩散罩，杜绝"灯带直贴塑料"的发黄陷阱
3. **可演示** —— 纯静态网页，无 build step，GitHub Pages 直接打开
4. **可制造** —— 输出 BOM 估算 + 装配步骤，让 1:1 复刻成为可能

## 关键风险

| 风险 | 影响 | 应对 |
|------|------|------|
| 散热不足导致外壳发黄 | 高 | LED 必贴铝槽，外壳不接触 LED |
| 扩散罩选错导致眩光 | 中 | 乳白 PC 扩散片，遮光角 ≥ 60° |
| 装配公差累积 | 中 | M3 法兰 + 卡扣双固定 |
| 颜色一致性 | 低 | 限定 4 色 SKU，Pantone 色卡对照 |
| 电源模块藏不进外壳 | 中 | 外壳底部留 ≥ 30mm × 50mm × 25mm 空间 |

## 跟现有方案对比

| 维度 | 网红氛围灯 | 传统阅读灯 | 本方案 ReadingCore-01 |
|------|----------|-----------|----------------------|
| 结构 | 灯带贴塑料 | E27 灯泡 + 反射罩 | 线性 LED + 铝槽 + 扩散罩 |
| 散热 | 差 | 好 | 优 |
| 显色 | Ra 80–90 | Ra 80–95 | Ra ≥ 95 |
| 可定制 | 颜色 | 无 | 风格 + 颜色 + 刻字 |
| 制造工艺 | 注塑 | 金属车件 | 3D 打印 + 通用铝型材 |
| 适合阅读 | 否 | 是 | 是 |
