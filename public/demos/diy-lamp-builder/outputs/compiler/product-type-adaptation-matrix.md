# Product Type Adaptation Matrix

| Product Type | Fixed Core | Variable Shell | First Validation | Risk Notes |
|--------------|------------|----------------|----------------|------------|
| Reading Lamp | ReadingCore-01 | lamp head / base / cultural shell | lux, heat, glare | electrical and heat boundaries |
| Night Light | LampCore-01 | translucent city / architecture shell | low heat, diffusion | brightness and material temperature |
| Aroma Lamp | aroma/heater module | decorative shell | heat soak, airflow | thermal and material safety |
| Desk Organizer | tray/grid core | custom exterior | fit and usability | low safety risk |
| Toothpick Box | drawer core | decorative shell | drawer fit | food-contact assumptions |
| Smart Speaker Shell | speaker/board core | acoustic enclosure | acoustic fit | electronics and acoustics |
| Mini Display Stand | stand core | themed shell | stability | tipping and fit |
| Gift Light Box | LED panel core | outer scene shell | diffusion and heat | material and glare |

## Selection rule

Start with lower-risk products:

- no battery
- no mains voltage
- no food contact
- no children's toy positioning
- no structural load
- fixed purchased functional module
