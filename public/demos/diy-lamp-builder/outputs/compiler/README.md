# IDB-6I Idea-to-Physical-Product Compiler

IDB-6I abstracts DIY Lamp Builder into a reusable method for turning a raw product idea into a physical prototype pipeline.

## Core pipeline

Idea → Fixed Core → Variable Shell → Configurator → CAD Export → Print Validation → Physical Testing → Build Package → Sourcing → First Build Runbook

## Why this matters

DIY Lamp Builder is no longer just one lamp demo. It becomes a reference implementation for a broader pattern:

> Keep the risky functional core fixed; let the user customize the shell, form, theme, and surface.
